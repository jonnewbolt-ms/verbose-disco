# Step 01 - Find Whiteboard

- Identify the meeting name and date, or the topic and participants.
- Use the appropriate prompt below to locate the Microsoft Whiteboard.
- Record the whiteboard file name, link, and last modified date.

## Scenario 1: You know the exact meeting
Prompt:
> "Find the Microsoft Whiteboard file associated with the Teams meeting '[Meeting Name]' held on [Date]. Look for files with a matching title or files modified by the participants around that time. Once found, provide the file name and a link."

## Scenario 2: You only know the topic or participants
Prompt:
> "Find any Microsoft Whiteboard files related to '[Topic]' or modified by [Person Name] in the last [Time Period]. List the files found with their last modified dates."
