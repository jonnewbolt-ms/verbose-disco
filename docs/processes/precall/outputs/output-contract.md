# Output contract

Required outputs:
- Engagement export workbook saved to `docs/processes/precall/exports/engagements_YYYY-MM-DD.xlsx`
- Email draft saved to `docs/processes/precall/exports/engagement_update_YYYY-MM-DD.eml` (or .msg if Outlook COM is available)
- A short report to the user summarizing hygiene flags

Scratch staging (non-deliverable):
- Temporary collection data stored under `docs/processes/precall/scratch/`

Optional outputs:
- None
